# EasyExam-API
AI-powered test paper generator for teachers

AI-powered test paper generator for teachers who want to create professional exam papers without using Word.

## Features

- Create test papers manually or generate questions using AI (Google Gemini)
- Export to PDF and Word (.docx)
- 5 customizable templates
- Live A4 preview
- Custom student info fields (Full Name, Class, Date)
- Style customization (font, spacing, option layout, colors)
- Answer key generation
## Tech Stack

**Backend:** ASP.NET Core 9 Web API, Entity Framework Core, SQLite, QuestPDF, OpenXML  
**Frontend:** Next.js 14, TypeScript, Tailwind CSS  
**AI:** Google Gemini 2.0 Flash
